# My Love Plugins

Claude Code plugin marketplace for the My Love PWA project.

## MCP Servers

| Server | Description |
|--------|-------------|
| playwright | Browser automation via Playwright |
| typescript-docs | TypeScript documentation via GitMCP |
| supabase | Supabase MCP integration |
| vitest | Vitest test runner MCP |
| refactor-typescript | TypeScript refactoring tools |

## Install

```
/plugin marketplace add Sallvainian/my-love-plugins
/plugin install mcp-servers@my-love-plugins
```
